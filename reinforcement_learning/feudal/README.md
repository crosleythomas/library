This paper shows how to create a Q-learning managerial hierarchy in which high level managers learn how to set tasks to their sub-managers who, in turn, learn how to satisfy them. Sub-managers need not initially understand their managersâ€™ commands. They simply learn to maximise their reinforcement in the context of the current command. We illustrate the system using a simple maze task..
